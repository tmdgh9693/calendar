HWPX 서식 초안 설정

회의자료와 출장복명서용 HWPX 파일을 여러 개 등록해 두고, 저장할 때 사용할 초안을 골라 쓸 수 있습니다. 등록한 초안 목록은 로그인한 사용자와 공유되지만, 내가 선택한 초안은 내 브라우저에서만 바뀝니다. 출장복명서 기본 양식은 사진 최대 6장까지 HWPX에 삽입합니다.

파일을 고른 뒤 선택한 파일 등록을 누르세요. 등록 직후 현재 용도의 사용 템플릿으로 자동 선택됩니다.
